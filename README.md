## About

This is a Pomodoro Timer desktop app built with Python. The Pomodoro Technique is a time management method where you work in focused 25-minute sessions separated by 5-minute breaks. This app helps students stay productive by visually tracking each session with circle timer, playing a sound when time is up, and keeping count of how many sessions you have completed.

## Features
- 25 min work timer and 5 minute break timer
- Visual circle timer that shows time as it runs out
- Start, Pause, and reset
- Celebration message at the end of each work session

##What I learned 
This project taught me alot of what Python can do beyond just writing scripts. Using tkinter to build an application was something I never did before. It was interesting learning how to fit all the pieces together like the canvas, buttons, and the timer. I learned alot about program structure

##Challenges 
The most difficult part was getting the circle arc to work correctly. I understood visually what I wanted but didn't know how to code in the feature. I got help from my dad who suggested that I make the arc by using degrees and split the remaining time into fractions.
